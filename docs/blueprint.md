# **App Name**: Student Bazar

## Core Features:

- User Authentication and Profiles: Enable user sign-up, sign-in, and profile management with options to update personal information, college name, WhatsApp number, and avatar selection.
- Product Listing and Discovery: Allow users to list new products with details and browse all available items using search and category filters. Feature a section on the home page for the most recent listings.
- Seller Dashboard and Public Profile: Provide sellers with a dashboard to manage their products, mark items as 'Sold,' and access a public profile displaying their listings, rating, and reviews.
- Direct Contact via WhatsApp: Facilitate direct communication between buyers and sellers through WhatsApp with pre-populated messages for product inquiries.
- Review and Rating System: Enable logged-in users to post public feedback on seller profiles, including a 5-star rating and comments.
- Smart Category Suggestion: In the 'Add Product' form, use AI to analyze the product name and description and suggest a category, allowing the user to confirm or override the suggestion. This is an AI tool.
- Admin Dashboard for Product Management: Provide an admin interface with a view of all registered users and listed products, enabling the admin to remove any product listing.

## Style Guidelines:

- Background: Deep, dark blue-gray (#0B1A2A) for a premium dark-mode-first design.
- Primary/Accent: Vibrant gold (#FFC857) for buttons, links, and important visual highlights.
- Card Backgrounds: Slightly lighter shade of the main background (#10141A) to create depth and separation.
- Text: Off-white for main body text to ensure high readability against the dark background. Dark text color for labels on gold-colored buttons.
- Font: 'Poppins' (sans-serif) for all text, including headlines and body copy, to create a consistent, modern, and friendly aesthetic. Note: currently only Google Fonts are supported.
- Page Transitions: Content animates in with a subtle fade-in-up effect on load.
- Visual Cues: Main call-to-action buttons have a soft, looping pulse glow animation.
- Interactivity: Product cards have a hover effect with a slight vertical lift and intensified shadow.
- Fully responsive design for desktop, tablet, and mobile devices.
- Sticky header containing the site logo, primary navigation, and user account menu.
